# Venture OS

A lightweight operating system for managing validated ventures.

## Structure
- `projects/` — one folder per approved project
- `execution/` — portfolio-level tracking and reviews
- `templates/` — reusable project and prompt templates
- `archive/` — parked or finished material

## Workflow
1. Idea is validated in the Venture Analyst chat.
2. If approved, create a new folder in `projects/` from the project template.
3. Build the actual product in its own separate code repository.
4. Use this repo as the planning / coordination layer across ChatGPT, Gemini, and Claude.
