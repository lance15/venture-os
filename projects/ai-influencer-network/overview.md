# AI Influencer Network

## Goal
Create 3 AI influencers with connected personalities and storylines that publish short-form content and can be monetised over time.

## Target User
- short-form content consumers
- followers interested in character arcs and micro-drama
- brands looking for digital creator collaborations later

## Problem Solved
Creates a scalable content system that can publish consistently without relying on a human creator being on camera.

## Revenue Model
- affiliate links
- brand deals
- digital products
- paid communities later

## MVP Features
- 3 character identities
- visual style guides
- posting plan for TikTok and Instagram
- short interaction/story format
- first batch of posts

## Current Status
Active

## Next Milestone
Define the 3 characters and produce the first content batch.
