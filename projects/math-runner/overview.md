# Math Runner

## Goal
Launch a mobile endless runner with an educational edge where players answer math questions by choosing the correct gate while running.

## Target User
- children and students
- parents seeking educational games
- casual mobile players

## Problem Solved
Makes math practice more engaging by embedding it in fast gameplay.

## Revenue Model
- ads
- cosmetic skins
- premium unlocks
- no-ads purchase

## MVP Features
- endless runner core loop
- 3 answer gates per question
- score and streak system
- difficulty scaling
- basic UI and restart flow

## Current Status
Active

## Next Milestone
Playable prototype with one full run cycle and functioning answer gates.
